Okay so i remade minesweerper in the GDscript engine in Godot. All the assets were pulled from the real game so the credit is for those. If you don't know the objective of minesweeper
is to clear the whole board without touching any bombs and placing flags on them. There's a timer and a flag score at the top and in the middle you can restart the game
by clicking the smiley.

I made this game fully by myself with some ai usage for debugging (cause there were a lot of bugs 💀😭). The games tile system is also customizable so if you put it in godot you can 
change the bomb amount, the grid width and height.
